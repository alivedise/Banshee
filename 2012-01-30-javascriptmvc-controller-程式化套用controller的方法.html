遇到一個作類似module loader的狀況，
有三種module a, b, c 分別以三個controller構成：controller_a, controller_b, controller_c
原本分開初始化一個controller很簡單，就是醬：


``` js println 1+2+3 $(element).controller_a({option: value});
$(element).controller_b({option: value});
$(element).controller_c({option: value});
```

現在想用動態方式決定作某個controller的初始化動作，方法如下：

``` js
var controll = 'contoller_a';
$(element)[controll]({option: value});
```
reference: http://forum.javascriptmvc.com/topic/programmatically-initial-a-controller#32525000000807021
註記：測試過後發現這個方法也適用於jQuery原生的controller，並非javascriptMVC專屬。
