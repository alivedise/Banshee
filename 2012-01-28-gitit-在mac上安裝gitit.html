使用homebrew->cabal->gitit這條路

gitit的github頁面 晚點來翻譯這個readme

[shell]
brew update
brew install cabal-install
[/shell]

這中間會開始下載ghc－－haskell的直譯器。當然，是darwin版本。
gitit是用haskell寫的，haskell本身是一套function lauguage，早期開發目的是拿來做數學運算。
很肥的GHC裝好後(約600M)會遇到cabal-install編譯不過的問題，
此時再裝haskell-platform

[shell]
brew install baskell-platform
[/shell]

之後

[shell]
cabal update
cabal install pandoc -fhighlighting --reinstall
cabal install gitit
[/shell]
